# 📅 Remind Me Of

A modern, highly responsive personal productivity dashboard built to help users manage their daily tasks, track upcoming events, and stay organized. 

The application features an interactive custom calendar, a color-coded reminder system, and a robust to-do list manager—all wrapped in a clean, dark-accented UI with smooth animations. Currently, all data is synced directly to the browser's Local Storage for immediate, secure access.

## ✨ Features

- **Interactive Custom Calendar:** A dynamic grid calendar built with `date-fns`. It visually tracks upcoming reminders with color-coded dot indicators (mobile) and detailed badges (desktop).
- **Task Management System:** Split tables for active To-Do's and completed (Done) To-Do's. Easily mark tasks as finished, undo completions, or delete them.
- **Categorized Reminders:** Create reminders with specific dates, start/end times, and color-coded categories (e.g., Activity, Study).
- **Persistent Storage:** Data is automatically saved and synced with the browser's `localStorage`, ensuring you never lose your schedule on reload.
- **Responsive Navigation:** A modern, collapsible sidebar that seamlessly adapts from desktop to mobile screens.

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Styling:** Tailwind CSS
- **Build Tool:** Vite
- **Date Utility:** date-fns
- **Icons:** react-icons, lucide-react
